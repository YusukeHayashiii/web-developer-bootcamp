やっていただきたいこと

  • 以下のような簡単な電卓ツールを作っていただきたい。
  • 作り方は様々あると思いますので、各自で自由に設計していただいて問題ありません。
  • JSで電卓を作るサイトはたくさんあると思いますので、外部のサイトを参考にして書いていただいても問題ないです。
  • 一例コードは下にあるので、それを軸にアレンジしていただいても問題ないです。
  • （恥ずかしくない方は）最後にどんなものになったかを見せていただきたいです。

進め方 (イチから作る人)

  • まずはhtmlとCSSで電卓の型を作成してください。
    • 出力欄はDomを用いて操作するので、 id or class名はつけてください
  • 次にJSコードで電卓機能を追加してください。
    • 計算する際に、evalという関数があるので、これを使うとショートカットできます。(https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Global_Objects/eval#試してみましょう)
    • button or input要素のonClickという属性があるので、これを用いて、以下のようにすると良いと思います。
    <button onclick="関数名">/</button>

進め方(一例から変更する人)

  • まずはどこにどんなコードを書いているのかを理解するところから、始めましょう
  • 一例をもとに、CSSなどを変えてみてください

    <button onclick="appendToDisplay('/')">/</button>
</div>

<script>
    const display = document.querySelector('#display');
    console.log(display)

    function appendToDisplay(value) {
        display.value += value;
        console.log(display.value)
    }

    function clearDisplay() {
        display.value = '';
    }

    function calculate() {
        try {
            display.value = eval(display.value);
        } catch (error) {
            display.value = 'Error';
        }
    }
</script>
